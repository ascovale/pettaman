# Petta

Questo repository contiene dei **prototipi** realizzati con Unity per esplorare diverse idee di gioco.

## Progetti

### Petta2
Prototipo focalizzato principalmente sulla **grafica e le animazioni**. I modelli e le animazioni sono scaricabili da [Meshy](https://www.meshy.ai/), con la possibilità di aggiungerne molte altre in futuro.

### PettaRoma
Prototipo orientato alla **meccanica di gioco**, con comandi in stile Roblox. Creato per sperimentare e capire come funzionano i controlli e le interazioni di base.

---

> **Nota:** si tratta di prototipi sperimentali, non di prodotti finiti.
